### FAQ Section

**Q: Do guests need to download an app?**
A: No. QR code opens in mobile browserâ€”works on any phone.

**Q: What if we run out of cards?**
A: Order more cards anytime ($50 for 100 cards). Or reuse cards from checked-out guests (reassign token to new guest).

**Q: Can we customize recommendations per guest type?**
A: Yes. Use tags to filter: "family-friendly," "romantic," "vegetarian," etc.

**Q: Does it integrate with our PMS?**
A: Not yet (roadmap Q2 2026). For now, staff manually enters guest name (30 seconds).

**Q: What if a restaurant closes?**
A: Update recommendation library in mztape admin. All future cards use updated info.
