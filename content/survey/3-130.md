# Todoist - S1 Provider Profile

## Overview

Todoist is a cloud-based task management application that has been a leader in the productivity space since its launch. The GTD method inspired many of its key functions, including quick add and inbox capture. As of 2025, Todoist remains a top-rated app on major to-do list comparisons.

**Platform:** Cross-platform (Web, iOS, Android, Mac, Windows, Linux)
**Primary Market:** Individual users and teams
**GTD Support:** Native - designed with GTD principles

## GTD Methodology Support

### Five-Step GTD Workflow

Todoist supports the complete GTD workflow:

1. **Capture:** Quick inbox for capturing tasks that come to mind
2. **Clarify:** Natural language processing for task entry (e.g., "send invoice on May 3" or "water the plants every 4 days starting June 20 ending Sept 30")
3. **Organize:** Combination of projects and labels for organization
4. **Reflect:** Filters and views for review
5. **Engage:** Priority levels and due dates for execution

### GTD-Specific Features

- **Inbox:** Central capture location for all incoming tasks
- **Projects:** Organize tasks by outcome/project
- **Labels:** Context-based organization (GTD contexts like @home, @computer, @phone)
- **Filters:** Create custom views for different GTD lists (Next Actions, Waiting For, Someday/Maybe)
- **Natural Language:** Quick task entry with dates and recurrence
- **Recurring Tasks:** Full support for recurring actions

### Limitations

The David Allen Company (official GTD) recommends Pro or Business versions to get full GTD functionality. The Free version has significant limitations:
- Only 5 projects/lists total
- Limited functionality for comprehensive GTD implementation

## Key Features

### 2025 AI Update

**Built-in AI Assistant:**
- Helps prioritize tasks
- Suggests schedules
- Identifies blockers
- Learns from patterns to offer smart nudges
- Keeps workflow flowing

### Core Functionality

- Natural language task creation
- Projects and sections for organization
- Labels and filters for flexible views
- Priority levels (P1-P4)
- Due dates and recurring tasks
- Comments and file attachments
- Collaboration features
- Productivity tracking and visualizations

## Pricing (2025)

### Free Plan
- Natural language task creation
- **5 projects maximum** (major limitation for GTD)
- Up to 5 collaborators per project
- Basic features

### Pro Plan
- **$4/month** (billed annually) or $5/month (billed monthly)
- 300 projects
- 25 collaborators per project
- Reminders
- Labels and filters
- File uploads (100MB per file)
- Themes and productivity trends
- **Recommended tier for GTD implementation**

### Business Plan
- $6/month per user (billed annually)
- 500 projects per team member
- 50 collaborators per project
- Team inbox
- Admin and member roles

## Platform Availability

- **Web:** Full-featured web app
- **Mobile:** iOS and Android (native apps)
- **Desktop:** Mac, Windows, Linux (native apps)
- **Integrations:** Email, calendar, voice assistants (Alexa, Google Assistant, Siri)
- **Browser Extensions:** Chrome, Firefox, Safari

## Strengths

1. **Excellent GTD Support:** Native GTD features with official setup guide from David Allen Company
2. **Cross-Platform:** Works on virtually every device and operating system
3. **Natural Language:** Quick task entry makes capture effortless
4. **Affordable:** Pro plan at $4/month offers excellent value
5. **AI Assistant (2025):** Smart features for prioritization and scheduling
6. **Active Development:** Regular updates and new features
7. **Large Ecosystem:** Extensive integrations and third-party tools

## Weaknesses

1. **Free Tier Too Limited:** 5 projects makes GTD implementation nearly impossible on free tier
2. **Requires Setup:** GTD implementation requires understanding labels, filters, and projects
3. **No Offline AI:** AI features may require internet connection
4. **Subscription Model:** Ongoing cost (though relatively low)
5. **Less Visual:** Compared to tools like Notion, more text-focused

## GTD Implementation Verdict

**Rating: Excellent (9/10)**

Todoist is one of the strongest GTD implementations available. With official endorsement from the David Allen Company and a setup guide specifically for GTD, it's purpose-built for the methodology. The combination of projects, labels, and filters provides all the structure needed for a complete GTD system.

The main barrier is the limited free tier - serious GTD users will need the Pro plan ($4/month). The 2025 AI features add intelligent assistance without compromising the core GTD workflow.

**Best For:**
- GTD practitioners seeking official, well-supported implementation
- Cross-platform users who need access everywhere
- Users who value speed (natural language processing)
- Budget-conscious users ($4/month is competitive)

**Not Ideal For:**
- Users unwilling to pay for Pro tier
- Teams needing advanced project management features
- Users preferring visual/Kanban-style interfaces
- Users wanting local-first/offline-first tool

# Things 3 - S1 Provider Profile

## Overview

Things 3 is a premium task management application developed by Cultured Code, designed around principles from the Getting Things Done (GTD) methodology. Launched in 2007 with a major redesign in 2017 (Things 3), it's known for its elegant design and Apple ecosystem integration.

**Platform:** Apple Ecosystem Only (macOS, iOS, iPadOS, Apple Watch)
**Primary Market:** Individual professionals (freelancers, designers, writers)
**GTD Support:** Strong - built with GTD principles, though not officially certified

## GTD Methodology Support

### GTD-Aligned Lists

Things 3 implements GTD concepts with its own terminology:

- **Inbox:** Capture location for new tasks (GTD: Inbox)
- **Today:** Tasks to start before day ends (GTD: Next Actions for today)
- **Upcoming:** Timeline view with scheduled tasks and deadlines (GTD: Calendar + Tickler)
- **Anytime:** Tasks that could be started anytime (GTD: Next Actions)
- **Someday:** Future possibilities (GTD: Someday/Maybe)

### Unique GTD Advantage

**Distinction Between Due Dates and "Do Dates":**

Things 3 is one of the few task tools that distinguishes between:
- **When (Do Date):** When you plan to START working on something
- **Deadline (Due Date):** When something must be FINISHED

This separation aligns perfectly with GTD's emphasis on next actions vs. hard deadlines.

### Organization Structure

- **Projects:** Group related tasks with clear goals
- **Areas:** Broader responsibility categories (Work, Personal, etc.)
- **Tags:** Flexible contexts, energy levels, time requirements
- **Headings:** Sub-organization within projects

### Calendar Integration

- View calendar appointments alongside tasks in Today and Upcoming views
- Calendar complements Things 3 (appointments live in calendar, tasks in Things)
- Visual timeline planning

## Key Features

### Core Functionality

- **Quick Entry:** Fast task capture with keyboard shortcuts
- **Natural Language:** Parse dates and times from task titles
- **Recurring Tasks:** Customizable repeating schedules
- **Checklists:** Subtasks and nested items
- **Notes & Attachments:** Add details to tasks
- **Tags:** Multi-tag support for flexible filtering
- **Today List:** Daily focus with evening review
- **Deadlines:** Separate from "when" dates

### Offline-First Design

- Fully functional without internet
- All data stored locally on device
- Automatic iCloud sync when online
- No dependency on cloud services

### Apple Ecosystem Integration

- iCloud sync (included free with purchase)
- Handoff between devices
- Widgets (iOS/macOS)
- Apple Watch app
- Share Sheet integration
- Siri shortcuts
- Keyboard shortcuts (Mac power users)

## Pricing (2025)

### One-Time Purchase Model

Things 3 uses a **one-time purchase per platform** (no subscription):

- **macOS:** $49.99
- **iPhone + Apple Watch:** $9.99
- **iPad:** $19.99

**Total for all platforms:** ~$80

### What's Included

- **Lifetime access** to purchased apps (no recurring fees)
- **Things Cloud sync:** Included free (no extra charge)
- **All features unlocked:** No premium tiers or add-ons
- **Family Sharing:** Share with family members via Apple Family Sharing
- **Free updates:** All future updates included

### Trial

- **15-day free trial** available for all platforms
- Full functionality during trial

## Platform Availability

### Supported

- macOS (native app)
- iOS (iPhone)
- iPadOS (iPad)
- Apple Watch

### NOT Supported

- ❌ Windows
- ❌ Android
- ❌ Linux
- ❌ Web app
- ❌ Cross-platform (Apple ecosystem only)

## Strengths

1. **Beautiful Design:** Widely praised for elegant, intuitive interface
2. **No Subscription:** One-time purchase, lifetime access
3. **Do Date vs Due Date:** Rare feature that aligns with GTD principles
4. **Offline-First:** Fully functional without internet
5. **Fast & Responsive:** Native apps optimized for Apple hardware
6. **Clean GTD Implementation:** Well-structured for GTD without being rigid
7. **Privacy-Focused:** Local-first data storage
8. **Included Sync:** Things Cloud sync at no extra cost
9. **Apple Ecosystem Integration:** Deep integration with macOS/iOS features
10. **Simplicity:** Easy to learn, not overwhelming

## Weaknesses

1. **Apple Ecosystem Only:** Cannot use on Windows, Android, or web
2. **High Upfront Cost:** $80 for full suite (vs $4/month for Todoist)
3. **No Collaboration:** Individual-focused, no team features
4. **Limited Integrations:** Fewer third-party integrations than competitors
5. **No AI Features:** Unlike Todoist 2025 AI, Things 3 has no intelligent assistance
6. **Closed Ecosystem:** Data export options limited
7. **Development Pace:** Updates less frequent than subscription-based competitors
8. **No Web Access:** Must use native apps

## GTD Implementation Verdict

**Rating: Excellent (9/10)**

Things 3 is one of the best GTD implementations available, particularly for Apple users. The distinction between "when" and "deadline" dates is a standout feature that few competitors offer. The clean interface makes GTD approachable without sacrificing power.

The lack of official GTD certification doesn't diminish its effectiveness - the structure (Inbox, Anytime, Upcoming, Someday) maps cleanly to GTD concepts. Tags handle contexts, Areas handle areas of responsibility, and Projects provide clear outcomes.

**Best For:**
- Apple ecosystem users (Mac + iPhone/iPad)
- Users preferring one-time purchase over subscription
- Privacy-conscious users wanting local-first storage
- Professionals valuing clean, distraction-free design
- Solo practitioners (not teams)
- Users wanting offline-first capability

**Not Ideal For:**
- Cross-platform users or non-Apple users
- Teams or collaborative work
- Users needing AI assistance
- Users wanting web access
- Budget-conscious users ($80 upfront vs $48/year Todoist Pro)
- Users needing extensive integrations

## Competitive Position

Things 3 occupies a premium position in the personal productivity space:

- **vs Todoist:** More expensive upfront, but no ongoing cost; Apple-only vs cross-platform; more polished design
- **vs OmniFocus:** Simpler, more approachable; less powerful but easier to learn; better value
- **vs Notion:** Focused tool vs all-in-one platform; better task management, worse for notes/databases
- **vs Obsidian:** Proprietary vs open; better task UX, worse for knowledge management

**Target User:** Professional Apple user willing to pay premium for elegant, focused GTD tool without subscription burden.

# OmniFocus - S1 Provider Profile

## Overview

OmniFocus was launched in 2008 by The Omni Group and designed specifically around the Getting Things Done (GTD) methodology from David Allen's book. It's the most powerful and complex GTD implementation available, built for power users who are deeply committed to the GTD system.

**Platform:** Apple Ecosystem + Web (limited)
**Primary Market:** GTD power users, knowledge workers, professionals
**GTD Support:** Excellent - purpose-built for GTD with official Omni Group GTD guide

## GTD Methodology Support

### Official GTD Implementation

OmniFocus has an official relationship with the GTD methodology and provides a comprehensive guide: "OmniFocus, GTD, and You: Implementing the Getting Things Done®" available from The Omni Group.

### GTD Workflow Support

**Complete GTD Structure:**
- **Inbox:** Universal capture for all incoming items
- **Projects:** Outcomes requiring multiple steps
- **Folders:** Group related projects (Areas of Responsibility)
- **Contexts/Tags:** Where/how/with whom actions can be done
- **Review Mode:** Structured review process for GTD Weekly Review
- **Perspectives:** Custom views for different GTD lists (Next Actions, Waiting For, Someday/Maybe)
- **Forecast View:** See tasks alongside calendar events (like GTD's Calendar + Tickler)

### Advanced GTD Features

- **Sequential vs Parallel Projects:** Control task availability
- **On Hold Status:** For projects/tasks not ready to start
- **Defer Dates:** Hide tasks until ready (tickler file)
- **Due Dates:** True deadlines (GTD-compliant - only when truly due)
- **Estimated Duration:** Time-based context planning
- **Energy Levels:** Flag tasks by required energy (via tags)
- **Review Intervals:** Customizable per project/area

## Key Features

### Power User Capabilities

**Custom Perspectives (Pro Edition):**
- Create filtered views for any GTD scenario
- Multiple criteria (tags, defer dates, project, flagged, etc.)
- Saved for quick access
- Examples: "Next Actions @work", "Waiting For", "High Energy Available"

**Review Mode:**
- Structured review interface
- Set review intervals per project/area
- Track last review date
- Prompts for GTD Weekly Review discipline

**Forecast View:**
- Timeline of upcoming tasks
- Integrated calendar events
- See commitments and deadlines together

**AppleScript Support (Pro):**
- Automation possibilities
- Integration with other tools
- Custom workflows

### Organization Tools

- **Projects & Folders:** Hierarchical organization
- **Multiple Tags:** Assign several contexts per task
- **Flags:** Mark high-priority items
- **Notes & Attachments:** Rich task details
- **Defer & Due Dates:** Separate start and deadline dates
- **Repeating Tasks:** Complex recurrence patterns

## Pricing (2025)

### Mac Version (One-Time Purchase)

- **Standard Edition:** $49.99 (one-time)
  - Core GTD features
  - Single platform (Mac only)

- **Pro Edition:** $99.99 (one-time)
  - Custom perspectives
  - AppleScript automation
  - More powerful for serious GTD users

### iOS/Mobile (Subscription)

- **$9.99/month** or **$99.99/year**
- Full access to iOS app features
- iPhone, iPad

### Cross-Platform Subscription

- **$9.99/month** or **$99.99/year**
- OmniFocus Pro on Mac, iPhone, iPad, and web
- Most flexible option for multi-device users

## Strengths

1. **Most Powerful GTD Implementation:** Every GTD concept has explicit support
2. **Custom Perspectives:** Create any filtered view imaginable

4. **Official GTD Guide:** Comprehensive documentation from Omni Group

**Rating: Exceptional for Power Users (10/10), Challenging for Beginners (6/10)**

OmniFocus is the most comprehensive GTD implementation available. If you're deeply committed to GTD methodology and need every advanced feature, this is the gold standard. The combination of custom perspectives, review mode, and flexible organization provides unmatched GTD power.

However, this power comes at a cost: complexity and price. The learning curve is steep, and casual users will be overwhelmed. This is a tool for GTD practitioners who have read David Allen's book, understand the methodology, and want software that matches their discipline.

**Best For:**
- Committed GTD practitioners
- Power users comfortable with complexity
- Apple ecosystem users

## Strategic Insights

### 1. Market Segmentation

**Tier 1: Purpose-Built GTD (Todoist, Things 3, OmniFocus)**
- 30-40% of personal productivity market
- Users committed to GTD methodology
- Willing to pay for specialized tools
- Premium pricing sustainable

**Tier 2: Flexible Platforms (Notion, Obsidian)**
- 40-50% of personal productivity market
- Users want tasks + other features (notes, wikis)
- DIY approach acceptable
- Mix of free and freemium

**Tier 3: Note Apps (Bear, Apple Notes, etc.)**
- NOT GTD market
- Simple checklists only
- Different user segment

### 2. Platform Ecosystem Divide

**Cross-Platform Strategy (Todoist, Notion, Obsidian):**
- Capture entire market (Windows/Mac/Linux/iOS/Android)
- Cloud-based sync required
- Vendor lock-in via cloud storage
- Subscription models dominate

**Apple Ecosystem Strategy (Things 3, OmniFocus, Bear):**
- 50% of tools are Apple-only
- Target premium segment (Apple users willing to pay)
- iCloud sync integration
- Mix of one-time and subscription

**Market Reality:** Windows + Android users have limited options (only 3 viable GTD tools).

### 3. Pricing Model Evolution

**Subscription Fatigue:**
- Average user: $50-60/month in subscriptions (Netflix, Spotify, Dropbox, etc.)
- Adding $4-10/month productivity tools contributes to subscription burden

**One-Time Purchase Appeal:**
- Things 3's "$80 forever" has psychological advantage
- No ongoing mental burden
- However, Things 3 is exception - most tools moving to subscriptions

**Freemium Winners:**
- Obsidian: Free forever, optional $8/month sync
- Notion: Free tier adequate for personal use
- Competitive pressure from free tiers pushes paid tools to justify value

### 4. AI Integration Opportunity

**Current State (2025):**
- Only Todoist has native AI (just launched)
- Features: Prioritization, scheduling, blocker detection, pattern learning

**Market Gap:**
- No AI-native GTD tools exist
- Opportunity for LLM-powered productivity:
  - Voice-first GTD (natural conversation)
  - Context-aware suggestions (location, time, energy)
  - Intelligent capture (understand intent from voice/email/notes)
  - Automatic project/context assignment
  - Smart weekly review (highlight what needs attention)

**Future Trajectory:**
- AI will become table stakes for productivity tools
- Todoist has 1-2 year head start
- Expect competitors to add AI within 12-18 months

### 5. Data Ownership Trend

**Local-First Movement:**
- Obsidian leads with markdown files
- Things 3 stores locally + iCloud
- Growing concern about cloud vendor lock-in

**Proprietary Cloud:**
- Todoist, OmniFocus, Notion store in proprietary formats
- Export possible but limited
- Switching costs high

**Market Demand:**
- Technical users increasingly value data ownership
- Mainstream users prioritize convenience over portability
- Niche but growing segment for local-first tools

# GTD Methodology Alignment Analysis

**How Well Does Each Tool Support Getting Things Done?**

**Reference:** David Allen's "Getting Things Done" (GTD) methodology

---

## GTD Methodology Overview

**The Five Steps:**
1. **Capture:** Collect what has your attention
2. **Clarify:** Process what it means
3. **Organize:** Put it where it belongs
4. **Reflect:** Review frequently
5. **Engage:** Simply do

**Key GTD Concepts:**
- Inbox (capture everything)
- Next Actions (very next physical action)
- Projects (outcomes requiring `>1` action)
- Contexts (@home, @computer, @phone, @errands)
- Waiting For (delegated items)
- Someday/Maybe (future possibilities)
- Weekly Review (critical habit)
- Calendar (hard commitments only)
- Reference (support material)

## 1. Individual vs Team GTD

### Current Market: Individual-First

**Individual-Only Tools (No Real Team Features):**
- Things 3 ❌
- OmniFocus ❌
- Bear ❌
- Obsidian ❌ (limited)

**Individual-Primary with Team Add-Ons:**
- Todoist ✅ (Business tier $6/user/month)
- Notion ✅ (designed for teams, works for individuals)

**Market Insight:** 67% of analyzed tools (4/6) have NO team collaboration features.

### When Personal Tools Become Insufficient

**Threshold Indicators:**

1. **Team Size:** 3+ people collaborating on shared projects
   - Personal tools: Solo or 1-2 occasional collaborators
   - Team tools: 3-20 people needing real-time collaboration

2. **Delegation Volume:** `>10` delegated tasks/week
   - Personal "Waiting For" lists sufficient up to ~10 items
   - Beyond that, need shared task assignment

3. **Project Complexity:** Multi-person projects with dependencies
   - Personal GTD: All tasks self-contained or simple delegation
   - Team GTD: Task dependencies, parallel work streams, shared deadlines

4. **Reporting Needs:** Team progress visibility
   - Personal: Only you need to see status
   - Team: Manager/stakeholders need visibility

**Transition Point:**
- **Individual → Team:** Typically at 5-10 collaborators or $500K+ revenue (small business)
- **Tools:** Migrate from Things 3/Todoist Pro → Asana/ClickUp/Linear (separate research category 3.131)

### Hybrid Approach

**Popular Pattern:** Personal GTD + Team PM
- **Personal:** Things 3 or Todoist (individual tasks, personal projects)
- **Team:** Asana or Linear (shared team projects)
- **Reference tasks** from team PM in personal GTD

**Example:**
- Marketing manager uses Linear for team sprint planning
- Uses Things 3 for personal tasks: 1:1 prep, email follow-ups, personal goals
- Linear tasks create Things 3 entries: "Review design mockups from team"

**Cost:** $80 Things 3 + $0-10/month team tool = manageable

## 2. Ecosystem Lock-In Analysis

### Platform Lock-In Risk Assessment

| Platform | Lock-In Level | Risk Factors | Mitigation |
|----------|---------------|--------------|------------|
| **Things 3** | 🔴 High | Apple-only, proprietary format | Export to JSON, switch to Todoist if leaving Apple |
| **OmniFocus** | 🔴 High | Apple-only, proprietary format | Export, re-import takes effort |
| **Todoist** | 🟡 Medium | Cloud-dependent, but cross-platform | CSV export, many import tools exist |
| **Notion** | 🟡 Medium | Proprietary database format | Markdown export, loses some structure |
| **Obsidian** | 🟢 Low | Plain markdown files | Copy files to any system |
| **Bear** | 🟠 Medium-High | Apple-only, export limited | Markdown export, re-import needed |

### Apple Ecosystem Lock-In

**Investment Required to Access Things 3/OmniFocus/Bear:**
- MacBook Air: $1,099+
- iPhone: $799+
- iPad: $349+
- **Total:** $2,247+ hardware investment

**Sunk Cost Fallacy:**
Users with $80-100 invested in Things 3/OmniFocus resist switching because:
1. Platform familiarity (months/years of muscle memory)
2. Data migration effort (4-8 hours)
3. Feature parity (Todoist missing when/due distinction, OmniFocus missing Review Mode)

**Strategy for Apple Users:**
- ✅ **Accept lock-in** if staying in Apple ecosystem long-term (95% of Mac users remain)
- ✅ **Choose Things 3** over subscriptions (minimize ongoing costs)
- ✅ **Avoid OmniFocus** unless truly need power features (less lock-in risk with $80 Things vs $1000+ OmniFocus over 10 years)

**Strategy for Cross-Platform Users:**
- ✅ **Never choose Apple-only tools**
- ✅ **Todoist or Obsidian** only safe long-term bets
- ✅ **Data export discipline** (quarterly backups)

### Cloud Vendor Lock-In

**Risk:** Todoist, Notion, OmniFocus store data in proprietary cloud

**Failure Scenarios:**
1. **Company shutdown** (unlikely for established players, but has happened)
2. **Acquisition + shutdown** (Wunderlist → Microsoft → killed)
3. **Price increase** (Todoist: $3→$4/month in 2023, 33% increase)
4. **Feature removal** (free tier degradation)
5. **Terms change** (AI training on user data, privacy policy changes)

**Mitigation Strategies:**

**For Cloud Tools (Todoist, Notion):**
- Weekly automated exports (JSON/CSV/Markdown)
- Backup to local storage or Git
- Have tested import path to alternative tool
- Budget for 2-3× price increases over 10 years

**For Local-First Tools (Obsidian, Things 3):**
- Multiple backup locations (iCloud + local + external drive)
- Things 3: JSON export monthly (just in case)
- Obsidian: Git version control (built-in)

### Data Portability Reality Check

**Export Quality by Platform:**

| Platform | Export Format | Quality | Re-Import Effort |
|----------|---------------|---------|------------------|
| **Todoist** | CSV, JSON | Good | 2-4 hours (many import tools) |
| **Things 3** | JSON | Good | 4-6 hours (fewer import tools) |
| **OmniFocus** | Custom | Medium | 6-10 hours (limited compatibility) |
| **Notion** | Markdown, CSV | Good | 6-10 hours (database structure lost) |
| **Obsidian** | Native Markdown | Excellent | 1-2 hours (or instant copy/paste) |
| **Bear** | Markdown | Good | 2-4 hours |

**Reality:** Even with export, expect to lose:
- Custom views/filters
- Automation/integrations
- Muscle memory/workflows
- Some metadata (tags, contexts may need remapping)

**True switching cost:** 10-20 hours (export + import + relearn + reconfigure)

### Historical Evolution

**2008-2015: GTD Golden Age**
- OmniFocus launches (2008)
- Things (2007) → Things 2 (2012)
- Todoist launches (2007)
- **Trend:** Purpose-built GTD tools proliferate

**2016-2020: All-in-One Platforms Rise**
- Notion launches (2016), grows rapidly
- Obsidian launches (2020)
- **Trend:** Users prefer unified workspaces over point solutions

**2021-2024: Subscription Fatigue**
- Things 3 resists subscription model (competitive advantage)
- Users seek one-time purchases or free options
- **Trend:** Backlash against subscription proliferation

**2025: AI Era Begins**
- Todoist adds AI assistant
- Notion AI launched (2023, expanded 2024-2025)
- **Trend:** AI integration becoming expected feature

### Current Market Dynamics

**Market Size:**
- Personal productivity software: $4B+ annually (estimate)
- GTD-specific subset: ~$500M-1B
- Growing 5-10% annually

**Competitive Landscape:**

**Leaders:**
- Todoist: Estimated 30M+ users (largest player)
- Notion: 30M+ users (all-in-one workspace, includes tasks)
- Microsoft To-Do: Bundled with Office 365 (large but low engagement)

**Premium Niche:**
- Things 3: ~1-2M users (Apple ecosystem, high revenue per user)
- OmniFocus: ~500K users (power users, high revenue per user)

**Open Source/Free:**
- Obsidian: ~5M users (free tier, growing fast)
- Org-mode (Emacs): Small but dedicated user base

**Market Concentration:**
- Top 3 (Todoist, Notion, Microsoft To-Do): 70%+ market share
- Long tail of niche tools: 30%

### Emerging Trends

**1. AI-First Productivity**
- **Examples:** Motion, Reclaim.ai (separate research: 3.133)
- **Trend:** New entrants are AI-native, not bolting AI onto existing tools
- **Implication:** Legacy tools (Things 3, OmniFocus) at risk if slow to add AI

**2. Privacy Backlash**
- **Drivers:** Big tech data scandals, GDPR, California Privacy Act
- **Trend:** Growing segment wants local-first, open-source tools
- **Beneficiaries:** Obsidian (fastest-growing in this segment)

**3. Platform Consolidation**
- **Examples:** Notion (docs + tasks + wikis), Obsidian (notes + tasks)
- **Trend:** Users prefer fewer, more capable tools vs. many point solutions
- **Implication:** Pure task managers (Things 3, OmniFocus, Todoist) must expand or risk obsolescence

**4. Mobile-First**
- **Driver:** Smartphones primary computer for many users
- **Trend:** Tools must have excellent mobile experience
- **Winners:** Things 3 (iPhone/iPad focus), Todoist (mobile-optimized)
- **Losers:** Desktop-heavy tools

**5. Subscription Resistance**
- **Countercurrent:** Despite industry push to subscriptions, users resist
- **Evidence:** Things 3 competitive advantage with one-time pricing
- **Trend:** Freemium + generous free tiers (Notion, Obsidian) OR one-time (Things 3)

## 5. Individual vs Team Strategic Decision

### Market Segmentation

**Individual Task Management Market (3.130):**
- Solo practitioners, freelancers, students
- Personal task management
- 1-2 occasional collaborators maximum
- **Tools:** Todoist, Things 3, OmniFocus, Notion, Obsidian

**Team Task Management Market (3.131 - separate research):**
- Small teams 3-20 people
- Shared projects and collaboration
- Permission models, team dashboards
- **Tools:** Asana, ClickUp, Linear, Monday.com, Trello

**Crossover Users:**
- ~30% of knowledge workers need BOTH
- Personal tasks + team projects
- **Strategy:** Use both (personal GTD + team PM)

### When to Graduate from Individual to Team Tools

**Signals:**

1. **Delegation Breakdown:** Spending `>2` hours/week managing "Waiting For" manually
2. **Visibility Requests:** Teammates/manager asking for status updates frequently
3. **Project Complexity:** Dependencies between team members difficult to track in personal tool
4. **Team Size:** 5+ people collaborating regularly

**Transition Strategy:**

**Phase 1: Individual Tool (0-3 collaborators)**
- Todoist Free/Pro, Things 3, Notion
- "Waiting For" list for delegated items
- Manual coordination via email/Slack

**Phase 2: Hybrid (3-10 collaborators)**
- Personal tool for individual tasks
- Team tool for shared projects
- Example: Things 3 (personal) + Asana (team projects)

**Phase 3: Team Tool Primary (10+ collaborators)**
- Team tool (Asana, ClickUp, Linear) for most work
- Personal tool for private tasks only
- Example: Linear (work) + Things 3 (personal life)

### Cost Analysis: Individual vs Team Tools

**Individual Tools (Annual Cost per User):**
- Obsidian: $0
- Things 3: $16/year (amortized over 5 years)
- Todoist Pro: $48/year
- OmniFocus: $100/year

**Team Tools (Annual Cost per User):**
- Trello Free: $0 (limited)
- Asana: $10.99/user/month = $131.88/year
- ClickUp: $7/user/month = $84/year
- Linear: $8/user/month = $96/year
- Monday.com: $10/user/month = $120/year

**Crossover Point:**
- For teams `>3` people, team tools ($84-132/user/year) become necessary
- Individual tools insufficient for collaboration
- ROI: Team tool saves `>10` hours/month in coordination = ~$500+/month value for $10-15/month cost
