## Research Approach

### 1. Provider Identification
Identify 5-6 major AI productivity tools across segments:
- **AI-First**: Motion, Reclaim.ai, Trevor AI, Akiflow
- **Guided Planning**: Sunsama, Structured
- **Comparison baseline**: Todoist, Things (non-AI)

### 2. Quick Evaluation Criteria
For each provider, capture:
- **AI sophistication**: Auto-scheduling vs manual vs hybrid
- **Pricing**: Monthly subscription cost
- **Key features**: Calendar integration, task sources, AI capabilities
- **Target user**: ADHD, managers, founders, knowledge workers
- **Differentiator**: Unique value proposition

### 3. Decision Framework
Create initial recommendation matrix:
- **Most advanced AI**: Which tool has best auto-scheduling?
- **Best value**: Price/performance ratio
- **Easiest to use**: Setup time, learning curve
- **Best for specific needs**: ADHD, managers, remote workers

## Provider Segments

### Segment 1: AI-First Tools

#### Motion
- **AI Level**: Most advanced (full auto-scheduling)
- **Pricing**: $34/month individual, $34/user/month teams
- **Key feature**: "Project Manager AI" - dependency-aware scheduling
- **Differentiator**: Fully automatic, hands-off scheduling
- **Target**: Knowledge workers, managers, teams with complex projects

#### Reclaim.ai
- **AI Level**: Advanced (habit + task scheduling)
- **Pricing**: Free (limited), $8-12/month Pro
- **Key feature**: "Habit AI" - recurring task optimization, buffer time
- **Differentiator**: Best calendar-first approach, generous free tier
- **Target**: Calendar-heavy users, meeting-intensive roles

#### Trevor AI
- **AI Level**: Medium (AI suggestions + manual override)
- **Pricing**: $9.99/month (was $3.99, increased 2024)
- **Key feature**: Visual time blocking with AI suggestions
- **Differentiator**: Manual control + AI assist (not fully automatic)
- **Target**: Visual thinkers, prefer manual control with AI help

#### Akiflow
- **AI Level**: Medium (task aggregation + AI scheduling)
- **Pricing**: $19/month (was $34/month, decreased 2024)
- **Key feature**: Universal inbox (aggregate all tasks), command palette
- **Differentiator**: Best task aggregation from multiple sources
- **Target**: Power users with tasks in multiple systems

---

### Segment 2: Guided Planning (Less AI)

#### Sunsama
- **AI Level**: Low (manual planning with light AI assists)
- **Pricing**: $20/month
- **Key feature**: Daily planning ritual (guided workflow)
- **Differentiator**: Mindful planning, not aggressive automation
- **Target**: People wanting structure, not full AI automation

#### Structured
- **AI Level**: Low (basic scheduling suggestions)
- **Pricing**: $14/month
- **Key feature**: Minimalist time blocking
- **Differentiator**: Simple, clean UI for basic time blocking
- **Target**: Minimalists, Apple ecosystem users

---

### Segment 3: Traditional (Non-AI Comparison)

#### Todoist
- **AI Level**: None (smart suggestions, no AI scheduling)
- **Pricing**: Free (basic), $5/month Premium
- **Key feature**: Fast task capture, natural language dates
- **Differentiator**: Best non-AI to-do list
- **Target**: People who just need task list, not AI scheduling

#### Things
- **AI Level**: None
- **Pricing**: $50 one-time (Mac/iPad), $20 (iPhone)
- **Key feature**: Beautiful Apple-native UI
- **Differentiator**: One-time purchase, no subscription
- **Target**: Apple users, dislike subscriptions

## Red Flags / Gotchas (S1 Observations)

### 1. **AI Calibration Period (2-4 Weeks)**
- All AI tools need time to learn your patterns
- First 2 weeks = mediocre suggestions
- After 4 weeks = personalized, accurate scheduling
- **Gotcha**: Don't judge tool quality in first week

### 2. **Calendar Clutter**
- AI schedules tasks as calendar events
- Can create 20-50 blocks/day (overwhelming)
- **Mitigation**: Only schedule priority tasks, keep admin off calendar

### 3. **Integration Complexity**
- More integrations = more complexity (Akiflow syndrome)
- Syncing Todoist + Asana + Linear + email = conflicts
- **Recommendation**: Start with 1 task source, expand gradually

### 4. **Price Increases**
- Trevor AI: $3.99 â†’ $9.99/month (2024, 2.5Ã— increase)
- Akiflow: $34 â†’ $19/month (2024, decreased to compete)
- Motion: Stable at $34/month
- **Observation**: Market still finding price equilibrium

### 5. **Mobile App Quality Varies**
- **Great**: Motion, Trevor, Akiflow (full-featured mobile)
- **Good**: Reclaim (mobile works, less powerful)
- **Meh**: Sunsama (mobile exists, desktop-first)
- **Critical**: Test mobile app before committing

### 6. **Team Features = 2Ã— Cost**
