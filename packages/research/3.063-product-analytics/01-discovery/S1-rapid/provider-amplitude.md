# Amplitude

## Description

Amplitude is an enterprise-grade product analytics platform that combines behavioral analytics with a customer data platform (CDP). It excels at custom event tracking, user segmentation, and predictive analytics powered by AI.

## Open Source Status

**Not open source** - Proprietary SaaS platform
**GitHub Presence**: 140+ repositories with SDKs (165 stars on TypeScript SDK)

## Pricing

**Starter (Free):**
- Up to 50k monthly tracked users (MTUs) or 10M events
- Core analytics features
- Unlimited feature flags
- Session Replay
- AI chat (beta)

**Plus:**
- Starts at $49/month (billed annually)
- 300k MTUs included
- Unlimited saved charts
- Behavioral cohorts
- Live chat support

**Growth:**
- Custom MTUs pricing
- Advanced behavioral analysis
- Causal insights & predictive audiences
- Real-time streaming
- Dedicated CSM access

**Enterprise:**
- Custom pricing
- Cross-product analytics
- Multi-armed bandits
- Warehouse-native analytics
- Advanced security (SCIM, EU data residency)

## Setup Time Estimate

**Time to First Event**: 25-35 minutes
- Comprehensive SDK library
- Requires more initial event planning than auto-capture tools
- Strong documentation and onboarding

## Key Differentiator

**Enterprise analytics with CDP integration**: Amplitude stands out with its integrated Customer Data Platform, enabling advanced segmentation and predictive analytics. The platform's AI-powered insights and causal analysis help teams move beyond basic metrics to understand why users behave certain ways, making it ideal for data-driven product teams.

## Best For

- Enterprise organizations with complex analytics needs
- Teams wanting CDP + analytics in one platform
- Companies requiring predictive analytics and AI insights
- Product-led growth companies tracking detailed user journeys
