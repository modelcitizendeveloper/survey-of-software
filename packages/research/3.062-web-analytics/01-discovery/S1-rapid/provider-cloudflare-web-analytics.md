# Cloudflare Web Analytics

**Category:** Privacy-focused / Developer-focused
**Website:** https://www.cloudflare.com/web-analytics/
**Pricing:** Free

## Key Features
- Completely free with no limitations
- No cookies, no localStorage, no client-side state
- No tracking of users over time (no fingerprinting)
- Available to all website owners (not just Cloudflare customers)
- Simple HTML snippet deployment

## Best For
Anyone wanting free, privacy-first analytics without any tracking or data collection concerns. Ideal for small sites and privacy-conscious projects.

## Quick Notes
- Enabled by default for Cloudflare free domains (can opt-out)
- Business model not based on user tracking
- Accessible via simple HTML snippet
- Very privacy-focused (considers fingerprinting "more intrusive than cookies")
- No paid tiers or upsells
