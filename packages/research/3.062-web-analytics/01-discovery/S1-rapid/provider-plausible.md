# Plausible Analytics

**Category:** Privacy-focused
**Website:** https://plausible.io/
**Pricing:** Starting at $9/month (up to 10K monthly pageviews)

## Key Features
- 75x smaller script than Google Analytics (~1KB)
- No cookies, no personal data collection
- Simple, single-page dashboard
- GDPR, CCPA, PECR compliant by default
- Open-source with self-hosting option available

## Best For
Privacy-conscious website owners, bloggers, and businesses operating in EU markets who want simple traffic insights without cookie banners.

## Quick Notes
- Over 15,000 paying subscribers
- Hosted on renewable energy servers in Germany
- Source code available on GitHub
- No data sold to third parties
- Growing alternative to Google Analytics in privacy-first markets
