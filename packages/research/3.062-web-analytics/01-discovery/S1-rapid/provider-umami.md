# Umami

**Category:** Self-hosted / Privacy-focused
**Website:** https://umami.is/
**Pricing:** Free (self-hosted) or Cloud free tier (1M events/month), then $20/month

## Key Features
- Lightweight script (~2KB) loads fast
- Simple setup and easy self-hosting
- GDPR/CCPA compliant by default
- Basic product analytics (user journeys, retention, funnels)
- Built with Next.js and React (modern stack)

## Best For
Developers and startups wanting a simple, self-hostable analytics solution with modern tech stack and minimal complexity.

## Quick Notes
- Self-hosted version is completely free with unlimited sites/events
- PostgreSQL or MySQL database
- Very popular in developer community for personal projects
- Modern UI and developer experience
- Growing feature set but still relatively basic compared to Matomo
