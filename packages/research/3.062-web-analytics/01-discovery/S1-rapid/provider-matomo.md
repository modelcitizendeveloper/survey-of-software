# Matomo

**Category:** Self-hosted / Full-featured / Privacy-focused
**Website:** https://matomo.org/
**Pricing:** Free (self-hosted) or Cloud starting at $23/month (50K hits/month)

## Key Features
- 100% data ownership and control
- Session recording and heatmaps (advanced features)
- Goals/conversions, e-commerce tracking, funnels
- GDPR, CCPA, PECR compliance built-in
- Rich feature set comparable to Google Analytics

## Best For
Organizations requiring data sovereignty, advanced analytics features, and full control over their analytics infrastructure. Popular in regulated industries.

## Quick Notes
- Originally launched as Piwik in 2007, most mature open-source analytics
- Trusted on 1M+ websites in 190+ countries
- Heavier client-side script (~23KB) compared to privacy-first alternatives
- Can be self-hosted (PHP/MySQL) or used as managed cloud service
- Most feature-rich privacy-focused option
