# S1 Synthesis: Personal Productivity Tools (GTD)
## Overview of 6 Platforms

**Research Date:** 2025-11-09
**Platforms Analyzed:** Todoist, Things 3, OmniFocus, Notion, Obsidian, Bear

---

## Executive Summary

The personal productivity/GTD space has **three tiers** of tools:

### Tier 1: Purpose-Built GTD Tools
- **Todoist, Things 3, OmniFocus**
- Native GTD features out of the box
- Ratings: 9-10/10 for GTD

### Tier 2: Flexible Platforms Adapted for GTD
- **Notion, Obsidian**
- Require setup via templates/plugins
- Ratings: 7-7.5/10 for GTD

### Tier 3: Note Apps with Basic Task Support
- **Bear**
- Not suitable for GTD
- Rating: 3/10 for GTD

## Key Findings

### 1. GTD Implementation Spectrum

**Explicit GTD (Purpose-Built):**
- Todoist: Official David Allen Company guide
- OmniFocus: Designed explicitly for GTD since 2008
- Things 3: Built with GTD principles (not certified)

**Template/Plugin GTD (Flexible):**
- Notion: Extensive GTD template ecosystem
- Obsidian: Community GTD plugins and vaults

**No Real GTD (Basic Tasks):**
- Bear: Simple checkboxes, not GTD-capable

### 2. Pricing Models

**Three distinct models:**

1. **Subscription:**
   - Todoist: $4/month ($48/year)
   - OmniFocus: $9.99/month ($99.99/year)
   - Notion Plus: $8/month ($96/year)

2. **One-Time Purchase:**
   - Things 3: $80 total (all platforms)
   - Bear: $14.99/year (hybrid)

3. **Free (with optional paid features):**
   - Notion: Free tier adequate for personal GTD
   - Obsidian: Free core + optional $8/month sync

**Value Analysis:**
- **Best value:** Notion Free or Obsidian Free
- **Best paid value:** Todoist Pro ($4/month)
- **Best no-subscription:** Things 3 ($80 one-time)

### 3. Platform Lock-In

**Cross-Platform (Work Everywhere):**
- Todoist ✅ (Windows, Mac, Linux, iOS, Android, Web)
- Notion ✅ (Windows, Mac, iOS, Android, Web)
- Obsidian ✅ (Windows, Mac, Linux, iOS, Android)

**Apple Ecosystem Only:**
- Things 3 ❌ (Mac, iOS, iPad only)
- OmniFocus ❌ (Mac, iOS, iPad + limited web)
- Bear ❌ (Mac, iOS, iPad only)

**Implication:** 50% of tools require Apple devices - significant lock-in risk.

### 4. Learning Curve

**Quick Start (< 1 hour):**
- Todoist: Natural language, intuitive
- Things 3: Clean UI, approachable

**Moderate (4-8 hours):**
- Notion: Learn databases and templates
- OmniFocus: Understand perspectives and review

**Steep (8-12+ hours):**
- Obsidian: Plugins, Dataview, markdown workflows
- OmniFocus: Full mastery for power users

### 5. GTD Feature Completeness

**Core GTD Requirements:**
- ✅ Inbox (capture)
- ✅ Projects (outcomes)
- ✅ Contexts (where/how)
- ✅ Next Actions (filtered views)
- ✅ Waiting For (delegated items)
- ✅ Someday/Maybe (future possibilities)
- ✅ Weekly Review (regular reflection)

### 6. Unique Differentiators

**Todoist:**
- Only tool with AI assistant (2025)
- Natural language processing
- Cross-platform leader
- Official GTD guide from David Allen Company

**Things 3:**
- Distinction between "When" (do date) and "Deadline" (due date)
- Most beautiful UI
- No subscription model
- Best Apple ecosystem integration

**OmniFocus:**
- Custom Perspectives (infinite GTD views)
- Review Mode (best weekly review support)
- Most powerful for complex workflows
- AppleScript automation

**Notion:**
- All-in-one: tasks + notes + databases + wikis
- Extreme flexibility
- Free tier adequate
- Visual options (Kanban, timeline, calendar)

**Obsidian:**
- Local-first, plain markdown files
- Complete data ownership
- Free forever
- Tasks + knowledge management unified

**Bear:**
- Cheapest ($14.99/year)
- Best writing experience
- Not suitable for GTD

## Strategic Insights

### Market Segmentation

**1. GTD Purists (10-15% of market)**
- Want explicit GTD features
- Willing to learn complex systems
- Target: OmniFocus

**2. GTD Practitioners (30-40% of market)**
- Want GTD-ready tools
- Prefer balance of power and ease
- Targets: Todoist, Things 3

**3. Flexible Users (40-50% of market)**
- Want customization + other features
- Willing to invest setup time
- Targets: Notion, Obsidian

**4. Casual Users (not GTD market)**
- Simple to-do lists
- Not targets: Bear, Apple Reminders, etc.

### Platform Ecosystems

**Closed Ecosystems:**
- Things 3, OmniFocus, Bear: Apple-only
- Todoist, Notion: Vendor lock-in to platform
- **Risk:** Loss of investment if switching platforms

**Open Ecosystems:**
- Obsidian: Plain markdown, portable
- **Advantage:** Future-proof, no lock-in

### AI Integration Landscape (2025)

**Current State:**
- **Only Todoist has native AI** (2025 update)
- Prioritization, scheduling, blocker identification
- Learns from patterns

**Others:**
- Notion AI exists but is separate add-on ($10/month extra)
- Things 3, OmniFocus, Obsidian, Bear: No AI

**Implication:** AI-powered GTD is nascent. Opportunity exists for AI-first GTD tools.

### Pricing Pressure

**Free Tiers Expanding:**
- Notion: Full GTD possible on free tier
- Obsidian: Completely free for personal use

**Low-Cost Leaders:**
- Todoist: $4/month competitive
- Bear: $14.99/year (but not GTD-capable)

**Premium Pricing:**
- OmniFocus: $99.99/year (10× Todoist)
- Things 3: $80 one-time (1.5× Todoist annual)

**Trend:** Pressure on premium tools to justify cost vs. free/cheap alternatives.

- Weakness: Expensive, steep learning curve, Apple-only

- Strength: Free, data ownership, markdown portability
- Weakness: Plugin complexity, DIY setup required

### Bear: **"The Note-Taking App (Not GTD)"**
- Strength: Beautiful writing, cheap
- Weakness: Not suitable for GTD

## Recommendations by User Profile

### "I need GTD that works on Windows + iPhone"
→ **Todoist** (only cross-platform purpose-built GTD)

### "I'm an Apple user who wants the best GTD experience"
→ **Things 3** (if prefer simplicity + no subscription)
→ **OmniFocus** (if need maximum power)

### "I want tasks + notes + wikis all in one place"
→ **Notion** (cloud) or **Obsidian** (local)

### "I care about data ownership and privacy"
→ **Obsidian** (plain markdown, local-first)

### "I want free forever"
→ **Notion Free** or **Obsidian**

### "I want AI-powered productivity"
→ **Todoist** (only option with native AI as of 2025)

### "I just need simple notes with occasional checklists"
→ **Bear** (but NOT for serious task management)

## Gap Analysis: What's Missing?

### 1. AI-First GTD
**Gap:** Only Todoist has AI (2025). No AI-native GTD tools exist.
**Opportunity:** AI-powered capture, prioritization, scheduling, context-aware suggestions

### 2. True Cross-Platform with Local-First
**Gap:**
- Cross-platform tools (Todoist, Notion) are cloud-dependent
- Local-first tools (Obsidian) require plugin complexity
**Opportunity:** Easy-to-use, cross-platform, local-first GTD

### 3. Frictionless Capture
**Gap:**
- Only Todoist has good natural language
- Others require manual date/context entry
**Opportunity:** Voice, email, chat-based GTD capture

### 4. Collaborative GTD
**Gap:** All analyzed tools are individual-focused
**Opportunity:** GTD for teams (separate category: 3.131)

### 5. Integration with LLMs
**Gap:** No tool deeply integrates with Claude, ChatGPT, etc.
**Opportunity:** LLM-powered GTD assistant that understands context
