## Feature Gap Analysis

### Missing from ALL Tools
- **LLM integration** (Claude, GPT-4) for intelligent task understanding
- **Voice-first GTD** (natural conversation for task management)
- **Context-aware suggestions** (based on location, time, calendar)
- **Collaborative GTD** (team-based GTD methodology support)
- **Energy-aware scheduling** (match tasks to energy levels automatically)

### Missing from MOST Tools
- **Do vs Due dates** (only Things 3 has this)
- **Review mode** (only OmniFocus has dedicated GTD review)
- **AI assistance** (only Todoist, as of 2025)
- **AppleScript automation** (only OmniFocus Pro)
- **Local-first with easy sync** (Obsidian has it, but requires setup)

## Conclusion

**No single tool wins everything.** The "best" tool depends on user needs:
