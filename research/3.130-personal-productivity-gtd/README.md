**Status:** ✅ Complete (2025-11-09)

## Overview

Comprehensive competitive analysis of individual task management and Getting Things Done (GTD) methodology tools. This research covers purpose-built GTD tools (Todoist, Things 3, OmniFocus) and flexible platforms adapted for GTD (Notion, Obsidian, Bear).

**Scope:** Individual/personal productivity tools (1-2 users)
**Out of Scope:** Team task management (see 3.131), enterprise project management (see 3.132)

---

## Executive Summary

### Top 3 GTD Tools (2025)

1. **Todoist Pro** ($48/year)
   - Best cross-platform GTD
   - Only tool with AI assistant (2025)
   - Natural language processing
   - **Best for:** Windows/Android users, AI features

2. **Things 3** ($80 one-time)
   - Best Apple ecosystem GTD
   - Beautiful design, no subscription
   - Unique when/deadline distinction
   - **Best for:** Apple users wanting simplicity

3. **OmniFocus** ($100/year)
   - Most powerful GTD
   - Only tool with Review Mode
   - Custom perspectives
   - **Best for:** Serious GTD practitioners

### Free Options

- **Obsidian** (free): Local-first, markdown, extensible
- **Notion** (free tier): All-in-one workspace

### Key Findings

1. **Platform Lock-In:** 50% of tools Apple-only (Things 3, OmniFocus, Bear) limits Windows/Android users
2. **AI Integration Nascent:** Only Todoist has native AI (2025), 12-18 month opportunity window
3. **GTD Methodology Support Varies:** Only OmniFocus has complete 5-step GTD with Review Mode
4. **Pricing Irrelevant:** $0-100/year cost negligible vs $7,200/year productivity value
5. **No Universal Winner:** Choice depends on platform, budget, technical skill, values

### Market Opportunity

**LLM-Powered GTD Tool:**
- Vikunja (open-source backend) + Claude AI interface
- Cross-platform, conversational task management
- Intelligent weekly review assistance
- 12-18 month window before competitors add sophisticated AI

## Quick Recommendations

### By Platform

- **Windows + Android:** Todoist Pro ($48/year)
- **Mac + iPhone:** Things 3 ($60-80 one-time)
- **Linux:** Obsidian (free)
- **Any platform:** Notion (free) or Obsidian (free)

### By Budget

- **$0:** Obsidian or Notion Free
- **$0-50/year:** Todoist Pro ($48) or Things 3 amortized ($16/year over 5 years)
- **$50-100/year:** OmniFocus ($100) for GTD experts

### By Use Case

- **GTD Beginner:** Things 3 (Apple) or Todoist (cross-platform)
- **GTD Expert:** OmniFocus (power features + Review Mode)
- **All-in-One Workspace:** Notion (tasks + notes + docs)
- **Privacy/Data Ownership:** Obsidian (local markdown)
- **AI Features:** Todoist (only option in 2025)
