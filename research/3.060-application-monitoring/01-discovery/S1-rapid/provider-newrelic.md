# Provider: New Relic

## Overview
New Relic is an established enterprise APM platform offering full-stack observability with consumption-based pricing. Known for its generous free tier and all-inclusive feature access.

## Popularity
- **Market Position**: Major player in APM market (alongside Datadog, Dynatrace)
- **Adoption**: Widely used in enterprise environments
- **Industry Recognition**: Frequently mentioned in APM market analyses

## Pricing (2025)
**Model**: Consumption-based (user + data)

**Free Tier**:
- 1 full platform user
- 100GB data ingestion/month
- All features included (no limitations)

**Standard Plan**:
- $49-99/user/month (varies by user count and host configuration)
- $0.30/GB for data above 100GB cap
- All platform features at every tier

**Cost Examples**:
- Small-scale: ~$500/month (2 full users + 500GB data)
- Mid-scale: $3,000-4,500/month (10 full users + 5TB data)

## Setup Time
**Estimated**: 15-30 minutes
- Agent installation on application servers
- SDK integration for detailed tracing
- Configuration of data collection and retention
- Dashboard setup and customization

## Key Differentiators
1. **Generous free tier** - Best free offering (1 full user + 100GB)
2. **All-inclusive features** - No feature gating by plan tier
3. **Consumption pricing** - Simpler model than modular competitors
4. **Established platform** - Mature product with extensive documentation
5. **Full-stack coverage** - APM, infrastructure, logs, synthetics, all included

## Best For
- Teams wanting to start free and scale gradually
- Organizations preferring simple consumption-based pricing
- Enterprises needing mature, battle-tested APM solution
- Projects requiring full-stack observability without multiple vendors
