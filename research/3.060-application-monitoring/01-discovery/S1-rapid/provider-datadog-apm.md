# Provider: Datadog APM

## Overview
Datadog is a comprehensive full-stack observability platform with Application Performance Monitoring (APM) as one of many integrated products. Known for deep infrastructure and application visibility.

## Popularity
- **Market Position**: Leader in application performance monitoring market
- **Adoption**: Enterprise-grade platform used by major organizations
- **Integration**: 600+ integrations with cloud, infrastructure, and application tools

## Pricing (2025)
**Model**: Modular per-product pricing

**APM & Continuous Profiler**:
- Base cost: $31/host/month (annual billing)
- APM for applications: +$31-45/host/month
- Each product has separate pricing logic

**Cost Examples**:
- Small-scale: ~$700/month (15 hosts with APM + infrastructure)
- Mid-scale: $4,600-6,000/month (100 hosts, multiple monitoring types)

**Note**: Costs can increase significantly with multiple products

## Setup Time
**Estimated**: 15-30 minutes
- Agent installation on hosts
- Configuration for APM tracing
- Integration with existing Datadog infrastructure monitoring
- More complex setup due to comprehensive feature set

## Key Differentiators
1. **Full-stack observability** - Unified platform for infrastructure, APM, logs, security
2. **Deep integrations** - 600+ integrations with modern stack components
3. **Enterprise-grade** - Built for large-scale, complex environments
4. **Advanced APM features** - Distributed tracing, service maps, profiling
5. **Unified dashboards** - Correlate application performance with infrastructure metrics

## Best For
- Large organizations needing comprehensive observability
- Teams already using Datadog for infrastructure monitoring
- Complex microservices architectures requiring distributed tracing
- Enterprises with budget for premium, integrated tooling
