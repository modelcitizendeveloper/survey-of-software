# Monday.com: Platform Profile

**Research Date**: November 12, 2025
**Profile Depth**: Full

---

## 1. Company Overview

**Founded**: 2012
**Founders**: Roy Mann (CEO), Eran Zinman (CTO)
**Status**: Public (NASDAQ: MNDY)
**Revenue**: $900M (2024)
**Headquarters**: Tel Aviv, Israel
**Employees**: ~2,000

### Market Positioning

"A Work OS that powers teams to run projects and workflows with confidence."

Monday.com positions itself as the **visual work operating system** - colorful, intuitive, flexible boards that anyone can use. Not just for project management, but for running all team workflows.

### Core Value Proposition

- **Visual**: Color-coded boards make work visible and engaging
- **Flexible**: Adapt to any workflow (not opinionated like Linear/Jira)
- **No-code**: Build complex workflows without technical skills
- **For everyone**: Non-technical teams (marketing, HR, sales) love it

## 2. Core Features

### Board-Based Work Management
- **Boards**: Everything is organized into visual boards
- **Groups**: Organize tasks within boards (collapsible sections)
- **Items**: Individual tasks/rows
- **Columns**: Custom fields (status, person, date, numbers, text, etc.)
- **Color-coding**: Visual status tracking (green = done, red = stuck)

### Column Types (30+ Types)
- **Status**: Color-coded dropdowns
- **Person**: Assign team members
- **Date**: Due dates, timelines
- **Numbers**: Budget, hours, quantities
- **Text**: Notes, descriptions
- **Dropdown**: Custom select lists
- **Timeline**: Start/end dates with visual bar
- **Files**: Attach documents
- **Link**: URLs to external resources
- **Formula**: Calculate values
- **Dependency**: Link related items
- **And 20+ more...**

### Multiple Views (15+ View Types)
- **Main Table**: Default board view
- **Kanban Board**: Drag-and-drop columns
- **Calendar**: Timeline of dates
- **Chart**: Gantt chart with dependencies
- **Map**: Geo-location view
- **Timeline**: Project scheduling
- **Workload**: Team capacity (Business+)
- **Dashboard**: Custom widgets
- **Form**: Collect requests
- **Files**: Gallery of attachments
- **And 5+ more...**

### Automation (Visual Workflow Builder)
- **Automations**: No-code rules ("When status changes to Done, notify person")
- **250-25,000 actions/month** depending on plan
- **Integrations**: Trigger automations from external apps
- **Recurring items**: Scheduled tasks

### Collaboration Features
- **Comments & mentions**
- **Activity log**: See all changes
- **File versioning**: Track document updates
- **Guest access**: External collaborators
- **Updates section**: Team communication hub

## 3. Team Collaboration

### Permission Models

| Role | Permissions |
|------|-------------|
| **Viewer** | Read-only access to boards |
| **Guest** | Limited access to specific boards (free, doesn't count toward seats) |
| **Member** | Create boards, items, collaborate |
| **Admin** | Manage users, account settings |
| **Owner** | Full control, billing |

### Board-Level Permissions
- **Public boards**: Everyone in workspace can see
- **Private boards**: Only invited members
- **Shareable boards**: Generate public links (view-only)
- **Guest access**: Invite external people (clients, contractors)

### Real-Time Collaboration
- Live updates across devices
- See who's viewing boards
- Notifications (customizable frequency)
- Mobile push notifications

## 4. Pricing Structure (2025)

### Free Plan
- **Cost**: $0
- **User limit**: 2 seats only
- **Features**:
  - Unlimited boards
  - Unlimited docs
  - 200+ templates
  - iOS & Android apps
  - 20 column types
  - 500MB storage
- **Limitations**:
  - Only 2 users (extremely limited)
  - No timeline/Gantt
  - No automations
  - No integrations

### Basic Plan
- **Cost**: $9/user/month (annual) or $12/user/month (monthly)
- **Minimum**: 3 users ($27/month minimum)
- **Features**:
  - Everything in Free, plus:
  - Unlimited items
  - 5GB storage
  - Priority support
  - Dashboard (1 board)
  - Mobile app
- **Limitations**:
  - No timeline/Gantt view
  - No automations (!)
  - No integrations (!)
  - Very basic

### Standard Plan
- **Cost**: $12/user/month (annual) or $14/user/month (monthly)
- **Minimum**: 3 users ($36/month minimum)
- **Features**:
  - Everything in Basic, plus:
  - Timeline & Gantt views
  - Calendar view
  - 250 automations/month
  - 250 integrations/month
  - Dashboard (up to 5 boards)
  - 20GB storage
  - Guest access
  - 6-month activity log
- **Best for**: Small teams (3-15 people) needing visual project management

### Pro Plan
- **Cost**: $19/user/month (annual) or $24/user/month (monthly)
- **Minimum**: 3 users ($57/month minimum)
- **Features**:
  - Everything in Standard, plus:
  - Private boards
  - Time tracking
  - Formula column
  - Dependency column
  - 25,000 automations/month
  - 25,000 integrations/month
  - Dashboard (up to 10 boards)
  - 100GB storage
  - 1-year activity log
- **Best for**: Growing teams (15-50 people) needing advanced features

### Enterprise Plan
- **Cost**: Custom pricing (contact sales)
- **Minimum**: 50+ users
- **Features**:
  - Everything in Pro, plus:
  - Single Sign-On (SAML, SCIM)
  - Advanced security & governance
  - Multi-level permissions
  - Enterprise-grade automations & integrations
  - 1TB storage
  - Dedicated account manager
  - Premium support (24/7)
  - Tailored onboarding
  - 99.9% uptime SLA
- **Best for**: Large organizations (50+ people) with security/compliance needs

### Cost Examples

| Team Size | Plan | Monthly Cost (Annual) | Annual Cost |
|-----------|------|----------------------|-------------|
| 3 users | Standard | $36 | $432 |
| 10 users | Standard | $120 | $1,440 |

## 8. Ideal Customer Profile

### Team Size Sweet Spot

**Best fit**: 5-50 people
- Large enough to justify cost
- Small enough to avoid complexity
- Cross-functional teams (not just engineers)

**Can work**: 3-100 people
- 3-5 people: Forced into 3-seat minimum, manageable
- 50-100 people: Works, but cost is high
- 100+ people: Enterprise plan (expensive)

**Not ideal**: 1-2 people or 200+ people
- 1-2 people: Free plan only 2 users, paid too expensive
- 200+ people: Better enterprise tools exist (Jira, Asana Enterprise)

### Industry / Use Case Fit

**Best for**:
1. **Marketing teams**: Campaign planning, content calendars, visual workflows
2. **Creative agencies**: Client projects, design workflows, approval processes
3. **Sales teams**: Pipeline management, lead tracking (CRM-lite)
4. **HR teams**: Recruiting, onboarding, employee workflows
5. **Operations teams**: Process management, visual dashboards
6. **Non-technical teams**: Anyone who prefers visual > text

**Not ideal for**:
1. **Software development**: Linear/Jira better for code-centric workflows
2. **Complex dependencies**: Gantt functionality weak
3. **Budget-conscious teams**: More expensive than ClickUp/Asana
4. **Tiny teams (1-2 people)**: Overkill and costly

### Technical vs Non-Technical Users

**Non-technical friendly**: ‚≠ê‚≠ê‚≠ê‚≠ê‚≠ê (5/5)
- Extremely visual and intuitive
- Color-coding makes status obvious
- No coding required (no-code workflows)
- Training time: 1-2 hours for basics

**Technical user perspective**:
- Developers often find it "too visual" (prefer Linear/Jira)
- API is decent (GraphQL)
- But not built for technical workflows
- Better for business users

### Key Decision Criteria

Choose Monday.com if you value:
1. **Visual/intuitive interface** (color-coded boards)
2. **Non-technical team** (marketing, HR, sales)
3. **Flexibility** (adapt boards to any workflow)
4. **Strong automation** (up to 25,000/month)
5. **Wide view variety** (15+ views)

Skip Monday.com if you need:
1. **Low cost** (ClickUp cheaper at $7/user)
2. **Generous free plan** (Asana 10 users, ClickUp unlimited)
3. **Developer workflows** (Linear/Jira better)
4. **Advanced Gantt** (MS Project, Smartsheet better)
5. **Simple tool** (Trello if you just need kanban)
