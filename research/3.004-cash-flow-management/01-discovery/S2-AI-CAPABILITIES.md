## Key Takeaways

### 1. Mosaic is Most Advanced (But Not Complete)
- **Covers:** ~60-70% of AI opportunities (Arc AI is market leader)
- **Gaps:** Narrative generation, Monte Carlo risk analysis, dynamic reforecasting
- **Verdict:** Even Mosaic users have 30-40% opportunity space

### 2. Mid-Tier Platforms (Jirav, Causal, Fathom) Cover 25-45%
- **Have:** Some AI features (Jirav budgeting, Causal integrations)
- **Missing:** NL interfaces, anomaly detection, advanced ML forecasting
- **Verdict:** 55-75% opportunity space remains

### 3. Small Business Platforms (Pulse, Finmark, Dryrun) Cover 10-30%
- **Have:** Basic forecasting and scenario modeling
- **Missing:** Nearly all advanced AI capabilities
- **Verdict:** 70-90% opportunity space

### 4. Universal Gaps (0-20% Coverage Everywhere)
- **Narrative report generation** - All platforms show data, none write summaries
- **Monte Carlo risk analysis** - Basic scenarios exist, probabilistic modeling doesn't
- **Dynamic reforecasting** - Platforms sync data, none auto-adjust forecasts
- **Cross-system AI intelligence** - Platforms integrate systems, but no AI analysis layer

### 5. Best Build Opportunities (High Value, Low Platform Coverage)

## Strategic Decision Tree: In-House AI Capabilities

This section helps you decide your strategy based on whether your organization has (or plans to build) in-house AI/ML capabilities.

---

### Path A: No In-House AI Capabilities (Most Organizations)

**Strategy: Maximize Platform AI, Accept Gaps**

#### Platform Selection Priority:
1. **Best AI coverage:** Mosaic (60-70% coverage, Arc AI)
2. **Good AI + affordability:** Jirav (35-45% coverage, $50-300/mo)
3. **Acceptable:** Causal, Fathom, Finmark (25-40% coverage)

#### What to Build vs Buy:
| Capability | Strategy | Solution |
|------------|----------|----------|
| **Cash flow forecasting** | Buy platform | Mosaic Arc AI, Jirav AI forecasts, or basic platform features |
| **Anomaly detection** | Buy platform (Mosaic only) | If not using Mosaic: manual review or third-party tools |
| **Variance analysis** | Buy platform | Mosaic Arc AI or manual analysis |
| **Scenario modeling** | Buy platform | All platforms support this |
| **Narrative reporting** | Accept gap or buy third-party | No platforms do this well - create templates |
| **Monte Carlo risk analysis** | Accept gap | Build in Excel/Python or hire analyst |
| **Benchmarking** | Buy platform (Fathom) or third-party | Look for industry-specific benchmark services |
| **Multi-system intelligence** | Buy integrations | Use platform integrations, manual analysis |

#### Decision Framework:
```
Q1: What's your budget for cash flow/FP&A software?
  <$100/mo ‚Üí Pulse or Fathom (basic AI = 10-35%)
  $100-$500/mo ‚Üí Jirav or Finmark (moderate AI = 35-45%)
  $500-$2,000/mo ‚Üí Mosaic or Causal (best AI = 60-70%)

Q2: Which AI capabilities matter most?
  Anomaly detection ‚Üí Mosaic (only option)
  AI budgeting ‚Üí Jirav
  Forecasting ‚Üí Mosaic or Jirav
  Multi-system integration ‚Üí Causal

Q3: For gaps, can you accept manual workarounds?
  Yes ‚Üí Choose platform, do narratives/Monte Carlo manually
  No ‚Üí Reconsider Path B or hire analysts
```

#### Action Plan:
1. Select AI-rich platform based on budget (Mosaic > Jirav > others)
2. Identify critical gaps (e.g., narrative reporting, Monte Carlo)
3. Build manual workarounds (Excel models, analyst reports)
4. Accept remaining gaps as unavailable until market matures

### Path B: Have In-House AI/ML Capabilities (Data Science Team, ML Engineers)

**Strategy: Use Platform for Basics, Build Custom AI for High-Value Gaps**

#### Platform Selection Priority:
1. **API access is critical** - prioritize platforms with robust APIs:
   - **Best APIs:** Causal (data warehouse connections), Mosaic, Jirav
   - **Good APIs:** Finmark, Fathom
   - **Limited APIs:** Pulse, Dryrun

2. **Don't overpay for platform AI** - if you're building custom, choose based on:
   - Data export capabilities
   - Integration breadth
   - Total cost (don't pay premium for Mosaic Arc AI if you're building your own)

#### What to Build vs Buy:
| Capability | Strategy | Why Build Custom |
|------------|----------|------------------|
| **Cash flow forecasting** | **Build custom** | Platform forecasting is basic, custom ML (Prophet, LSTM) far superior |
| **Monte Carlo risk analysis** | **Build custom** | 0% platform coverage, massive value, Python implementation straightforward |
| **Narrative reporting** | **Build custom** | 0% platform coverage, high ROI for board/investor relations |
| **Dynamic reforecasting** | **Build custom** | Platforms don't do this, critical for fast-moving companies |
| **Cross-system intelligence** | **Build custom** | Platforms integrate but don't analyze, LLM layer adds huge value |
| **Anomaly detection** | Build if not using Mosaic | Mosaic covers 70%, custom ML (Isolation Forest) superior |
| **Variance analysis** | Buy if using Mosaic, build otherwise | Mosaic Arc AI is good, custom can be better |
| **Scenario modeling** | Buy platform | Platforms do this well, not worth custom build |

#### High-ROI Build Opportunities (Ranked):
1. **ü•á Monte Carlo & probabilistic forecasting** (0% platform coverage, massive value)
   - Python libraries (NumPy, SciPy, Monte Carlo simulation)
   - Risk-adjusted cash runway ("30% chance of running out in 6 months")
   - Scenario probability weighting
   - Estimated ROI: Avoid cash crises, optimize fundraising timing

2. **ü•à Advanced time series forecasting** (30-50% basic platform coverage)
   - Time series ML models (Prophet, ARIMA, LSTM, XGBoost)
   - External factor integration (economic indicators, seasonality)
   - Confidence intervals and forecast ranges
   - Estimated ROI: 20-40% forecast accuracy improvement

3. **ü•â Narrative report generation** (0-10% platform coverage)
   - LLM auto-generates board deck summaries
   - Metric explanation ("Why did burn increase?")
   - Comparative analysis to benchmarks
   - Estimated ROI: 10-20 hours/month saved in report writing

4. **Dynamic reforecasting** (0% coverage)
   - Continuous forecast updates as actuals arrive
   - Anomaly-triggered reforecasts
   - Automated variance re-projection
   - Estimated ROI: Real-time visibility, faster decision-making

5. **Cross-system AI intelligence** (0% coverage)
   - ETL from accounting + payroll + CRM + revenue systems
   - Unified data warehouse with LLM query interface
   - Pattern detection across systems
   - Estimated ROI: 20-40 hours/month saved in manual analysis

- **Time Series Models:** Prophet, ARIMA, LSTM (TensorFlow/PyTorch), XGBoost
- **Risk Modeling:** NumPy, SciPy (Monte Carlo), PyMC3 (Bayesian)
- **Data Warehouse:** PostgreSQL, Snowflake, BigQuery
- **ETL:** Airbyte, Fivetran, or custom Python scripts
- **Visualization:** Plotly, Tableau, or custom dashboards

#### Decision Framework:
