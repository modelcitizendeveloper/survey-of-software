# Hotjar

**Category:** Qualitative Analytics (specialized)
**Website:** https://www.hotjar.com/
**Pricing:** Free (Basic), Plus $40/month, Business $100/month, Scale $213/month

## Key Features
- Heatmaps and click tracking visualization
- Session recordings for user behavior analysis
- Feedback polls and surveys
- Conversion funnel analysis
- User recruitment for testing

## Best For
UX teams and marketers who need to understand not just what users do, but why they do it through qualitative insights.

## Quick Notes
- Complements traditional analytics with qualitative data
- Session replay is key differentiator
- Free tier available for basic usage
- Best used alongside quantitative analytics tools
- Focus on user experience optimization
- Borderline scope (mentioned in S0 as "include only if bundled")
