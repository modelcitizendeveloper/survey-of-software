# Mixpanel

**Category:** Product Analytics (hybrid - borderline scope)
**Website:** https://mixpanel.com/
**Pricing:** Free tier, Growth from $20/month (100M events), Enterprise from $833/month

## Key Features
- Event-based analytics with unlimited tracking
- Advanced funnel analysis and cohort tracking
- User journey visualization
- Comprehensive data governance and security
- A/B testing and experimentation features

## Best For
Product teams and SaaS companies needing detailed event tracking and user behavior analysis beyond basic web analytics.

## Quick Notes
- Shifted from MTU to event-based pricing (2025)
- Startup program: first year free for early-stage companies (<$8M funding)
- More product analytics than web analytics
- Strong in mobile and app analytics
- Borderline out-of-scope per experiment definition but commonly considered for web analytics
