# Vercel Analytics

**Category:** Developer-focused / Platform-integrated
**Website:** https://vercel.com/analytics
**Pricing:** Included with Vercel Pro ($20/month) and Enterprise plans

## Key Features
- Native integration with Vercel deployment platform
- Core Web Vitals tracking and performance monitoring
- Real user monitoring (RUM)
- Privacy-friendly (minimal data collection)
- Zero configuration for Vercel-hosted sites

## Best For
Teams deploying on Vercel platform who want integrated performance and analytics without third-party tools.

## Quick Notes
- Tightly integrated with Vercel deployment workflow
- Focus on Core Web Vitals and performance metrics
- Limited compared to full analytics platforms
- Best as complementary tool for Vercel users
- Requires Vercel hosting platform
